# ModernMaps
This repository contains public maps used in the Casual Modern PGM servers.

Many of these maps were created and submitted by members of the Casual Modern PGM community headed by [zzuf](https://github.com/zzufx) and [Eliminate](https://github.com/eliminate123).

## Important

When using a map or include file from this server on another server, keep in mind the following points:

- Because of our modern-only setup, some XML lines might not be properly wrapped in version conditionals (`<if min-server-version="XX"> ... </if>`). This could cause unintended side effects on other server setups or versions. 
- Maps might use features from [our own PGM fork](https://github.com/CasualModernPGM/PGM), which will not work if using the upstream version.
